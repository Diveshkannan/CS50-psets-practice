# Hello Program in C

## Overview

This program demonstrates a simple greeting application written in C.  
It prompts the user for their name using the CS50 library’s `get_string` function and then prints a personalized greeting.

---

## Features

- Written in standard C with CS50 library support  
- Uses `get_string` for safe and convenient string input  
- Clean and minimal implementation suitable for beginners  
- Demonstrates integration of external libraries in C projects  

---

## Algorithm Concept

1. Prompt the user for their name.  
2. Store the input string.  
3. Print a greeting message using the provided name.

# 🖐️ Hello Program (CS50 Pset1)

This is the classic **Hello** program from CS50’s Problem Set 1.  
It asks the user for their name and greets them personally.

---

## 📜 Description
- Prompts the user: *“What is your name?”*  
- Reads the input using CS50’s `get_string`.  
- Prints a friendly greeting

## Example run
What is your name?David
Hello David

---

## ✨ Final Notes

This **Hello Program** may look simple, but it marks the beginning of your CS50 journey.  
Every great program starts with a single line of output — and this one teaches you the fundamentals of **input, output, and strings in C**.

- 🔹 It’s your first step into problem solving with code.  
- 🔹 It shows how even small programs can be powerful when they interact with the user.  
- 🔹 It sets the foundation for more complex challenges ahead.

Keep experimenting, keep learning, and remember:  
> *“From a single `hello`, a thousand programs are born.”*

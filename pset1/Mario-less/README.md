# 🏰 Pyramid Builder (CS50 Practice)

This program builds a **text-based pyramid** using hashes (`#`) based on the number of rows the user provides.  
It’s a step up from the classic “Hello” program — moving from a simple greeting to structured output.

---

## 📜 Description
- Prompts the user: *“Number of rows:”*  
- Reads the input using CS50’s `get_int`.  
- Prints a pyramid of `#` characters, aligned neatly with spaces.  
- Each row grows taller, forming a staircase-like pyramid.

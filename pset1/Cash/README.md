# 💰 Change Owed Calculator (CS50 Practice)

This program calculates the **minimum number of coins** needed to make change for a given amount of cents.  
It ensures the user enters a valid positive number before performing the calculation.

---

## 📜 Description
- Prompts the user: *“Change owed:”*  
- Validates input, asking again if the user enters zero or a negative number.  
- Breaks down the amount into **quarters (25¢), dimes (10¢), nickels (5¢), and pennies (1¢)**.  
- Outputs the total number of coins required to make the change.

---

## 💻 Example Run
Change owed: -5 
Please enter more than 0 
Change owed: 41
4


Explanation:  
- 41 cents → 1 quarter (25¢), 1 dime (10¢), 1 nickel (5¢), 1 penny (1¢).  
- Total coins = 4.

# 🏰 Validated Double Pyramid Builder (CS50 Practice)

This program builds a **text-based double pyramid** using hashes (`#`) based on the number of rows the user provides.  
Unlike earlier versions, it gives a clear warning when the user enters invalid input and keeps asking until a valid positive number is provided.

---

## 📜 Description
- Displays a message: *“Please enter a positive integer.”* whenever the user enters zero or a negative number.  
- Uses a `do...while` loop to validate input, ensuring the number of rows is greater than zero.  
- Prints two pyramids of `#` characters, aligned neatly with spaces and separated by a gap.  
- Each row grows taller, forming a mirrored staircase-like structure.

---

Number of rows:-1
Please enter a positive integer.

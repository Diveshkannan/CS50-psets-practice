# 🏰 Double Pyramid Builder (CS50 Practice)

This program builds a **text-based double pyramid** using hashes (`#`) based on the number of rows the user provides.  
It’s a creative extension of the single pyramid exercise — now printing two pyramids side by side with a space in between.

---

## 📜 Description
- Prompts the user: *“Number of rows:”*  
- Reads the input using CS50’s `get_int`.  
- Prints two pyramids of `#` characters, aligned neatly with spaces.  
- Each row grows taller, forming a mirrored staircase-like structure.

---

## 💻 Example Run
No of rows:4
   #  #
  ##  ##
 ###  ###
####  ####
